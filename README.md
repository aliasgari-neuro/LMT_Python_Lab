# LMT_Python_Lab
A community-driven library of Python solutions for linguistics and cognitive science, focusing on the intersection of language, creativity, and digital methodology.
Official code repository for the **Language, Mind & Technology** Python course at **Adam Mickiewicz University (AMU)**.

## 📂 Structure
- `/sessions`: Weekly lecture scripts and exercises.
- `/contributions`: Peer-reviewed student code snippets.
- `/data`: Sample datasets for linguistic analysis.

## ✍️ Contribution
1. Post your `.py` file in the Discord `#💻-code-snippets` channel.
2. Once reviewed by the instructor, it will be added to the `/contributions` folder.
3. Every approved contribution earns **+2 Extra Credit**.
